# Sakura-Player
USAGE Instructions:

 1. Clone the project:
      git clone https://github.com/Nota-not/Sakura-Player.git

   2. Install dependencies:
      npm install

   3. Create .env file with Spotify credentials 
      (Can use any text editor: Notepad, TextEdit, etc.)

   4. Get their own Spotify credentials from: https://developer.spotify.com/dashboard
Create their own .env file with their credentials

   5. Run the app:
      npm start

   6. App opens in a window → Test it → Report bugs

(eg. # 1. Clone your repo
git clone https://github.com/Nota-not/Sakura-Player.git

# 2. Install dependencies
npm install

# 3. Create .env in the project root with THEIR credentials:
SPOTIFY_CLIENT_ID=their_client_id_here
SPOTIFY_CLIENT_SECRET=their_client_secret_here
REDIRECT_URI=http://127.0.0.1:8888/callback

# 4. Run the app
npm start

)
