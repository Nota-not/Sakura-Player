# Sakura-Player
USAGE Instructions:

 1. Clone the project:
      git clone https://github.com/Nota-not/Sakura-Player.git

   2. Install dependencies:
      npm install

   3. Create .env file with Spotify credentials
      (Can use any text editor: Notepad, TextEdit, etc.)

   4. Run the app:
      npm start

   5. App opens in a window → Test it → Report bugs
